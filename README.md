# UnrealEngine_Base_Manual
Manuale per la creazione di un progetto in UnrealEngine

##Recommended Directory Tree in Unreal Content Drawer
```text
Content/
├── Blueprints/
│   ├── Characters                  #Contains Blueprints for Character behaviors
│   └── GameMode.uasset             #Blueprint in Unreal, Sets Game Rules
├── Fonts/                          #Contains Text Fonts
├── Input/
│   ├── Actions/                    #Contains all Input Actions
│   └── InputMappingCotext.uasset
├── Maps/                           #Contains the World Map
├── PaperAssets/
│   ├── Characters/                 #Contains Sprite, Texture, Sprite Sheet and Flipbook for each Character
│   ├── Environment/                #Contains Texture, Tile Map and Tile Set for each Envirorment
│   ├── Props/                      #Contains Sprite, Texture, Sprite Sheet and Flipbook for each Props
│   └── VFX/                        #Contains Sprite, Texture, Sprite Sheet and Flipbook for each VFX
├── Sounds/                         #Contains all the Sounds
└── UI/                             #Contains all the User Inerface
```
